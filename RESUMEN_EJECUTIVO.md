# 🚀 RESUMEN EJECUTIVO - Maket AI

## ✨ **En Una Sola Frase**
Maket AI es una plataforma que permite a cualquier emprendedor crear un negocio online completo (landing + 3 agentes IA especializados) **para CUALQUIER industria**, solo mediante conversación.

---

## 🌍 **Funciona para CUALQUIER Negocio**

| Industria | Ejemplo | Agente Vendedor | Capacidades Específicas |
|-----------|---------|----------------|------------------------|
| 🍔 **Restaurantes** | Hamburguesería | María | Toma pedidos, sugiere combos, personalización |
| 👗 **Moda/Ropa** | Tienda urbana | Sofía | Asesora tallas, crea outfits, combina prendas |

## 🤖 **Los 3 Agentes**

### **1. Orquestador (Constructor)** 🏗️
**Qué hace:** Crea el negocio completo conversando con el dueño (11 fases).

**Cómo se adapta:** Detecta automáticamente el tipo de industria y configura todo según ella.

**Ejemplo:**
```
Dueño: "Quiero vender cursos de programación"
Orquestador: Detecta → educacion, cursos_online
             Genera → Landing educativa
                      Prof. Ana (vendedor)
                      Max +  (admin)
                      Catálogo de cursos
```

### **2. Agente Vendedor** 💬
**Qué hace:** Atiende clientes, recomienda productos/servicios, cierra ventas.

**Cómo se adapta:** Usa plantilla específica de la industria + aprende de cada cliente.

**Ejemplos:**
- **Restaurante:** María toma pedidos y sugiere "¿Agregas papas por $2?"
- **Ropa:** Sofía pregunta "¿Qué talla usas normalmente?"
- **Tecnología:** Alex explica "Este modelo tiene 16GB de RAM vs 8GB de aquel"
- **Gym:** Coach Mike dice "Para tus objetivos, te recomiendo 3 días/semana"

### **3. Administrador (Max)** 📊
**Qué hace:** Gestiona inventario, optimiza precios, genera reportes, toma decisiones autónomas.

**Cómo se adapta:** Funciones específicas según tipo de negocio.

**Ejemplos:**
- **Restaurante:** Detecta stock bajo de ingredientes → orden de reposición
- **Ropa:** Detecta talla M agotándose → reposición + promoción en otras tallas
- **Tecnología:** Detecta modelo viejo en inventario → descuento automático
- **Gym:** Detecta clase con poca inscripción → crea promoción

## 🎯 **Flujo Universal**

```
┌──────────────────────────────────────────────────────────┐
│         PASO 1: Creación (con Orquestador)               │
└──────────────────────────────────────────────────────────┘

Dueño: "Quiero crear [tipo de negocio]"
   ↓
Orquestador (11 fases de conversación):
   1. Detecta tipo de industria automáticamente
   2. Selecciona plantillas apropiadas
   3. Configura identidad de marca
   4. Genera estructura de catálogo adaptada
   5. Crea reglas de dominio específicas
   6. GENERA Agente Vendedor especializado
   7. GENERA Agente Administrador especializado
   8. Configura aspectos comerciales
   9. Activa automatizaciones
   10. Muestra preview
   11. ACTIVA el negocio
   ↓
✅ RESULTADO: Negocio online operativo en 30 minutos

┌──────────────────────────────────────────────────────────┐
│         PASO 2: Operación (automática)                   │
└──────────────────────────────────────────────────────────┘

Cliente visita la tienda pública
   ↓
Chatea con Agente Vendedor (especializado)
   ├─ Carga perfil del cliente (si existe)
   ├─ Recomienda productos/servicios relevantes
   ├─ Responde preguntas específicas de la industria
   └─ Completa la venta
   ↓
Sistema automáticamente:
   ├─ Actualiza perfil del cliente (aprende preferencias)
   ├─ Descuenta inventario
   └─ Actualiza métricas
   ↓
Max (Administrador) cada hora:
   ├─ Analiza stock/inventario
   ├─ Detecta oportunidades (promociones)
   ├─ Toma decisiones autónomas
   └─ Genera reportes para el dueño
```

## 💡 **Diferenciadores Clave**

### **vs Otros Sistemas**

| Característica | Otros sistemas | Maket AI |
|---------------|----------------|----------|
| **Configuración** | Manual, días/semanas | Conversacional, 30 minutos |
| **Adaptación** | Generic templates | Experto por industria |
| **Agentes IA** | Un chatbot general | 3 agentes especializados |
| **Aprendizaje** | No aprende | Perfiles automáticos |
| **Decisiones** | Manual by dueño | Autónomas por Max |
| **Multi-negocio** | Limitado a 1 tipo | Cualquier industria |
| **Escalabilidad** | 1 negocio = 1 deploy | N negocios, 1 plataforma |

## 📊 **Casos de Uso Reales**

### **Caso 1: Emprendedor sin experiencia técnica**
```
Situación: Quiere abrir tienda de ropa pero no sabe programar
Solución: 
   1. Conversación de 30 min con Orquestador
   2. Sistema genera todo automáticamente
   3. Tienda online operativa
   4. Agente Sofía atiende clientes 24/7
   5. Max gestiona inventario por talla/color
```

### **Caso 2: Restaurante existente quiere digitalización**
```
Situación: Tiene restaurante físico, necesita presencia online
Solución:
   1. Orquestador configura menú digital
   2. María toma pedidos online
   3. Max gestiona ingredientes y alertas
   4. Sistema de delivery automatizado
```

### **Caso 3: Emprendedor serial (múltiples negocios)**
```
Situación: Quiere gestionar gym + tienda de suplementos + nutrición
Solución:
   1. Crea 3 negocios en la misma plataforma
   2. Cada uno con agentes especializados:
      - Coach Mike para gym
      - Alex para suplementos  
      - Nutrióloga Carmen para asesorías
   3. Dashboard único para ver todo
   4. Max gestiona los 3 negocios
```
